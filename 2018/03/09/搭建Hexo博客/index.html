发表于 2018-03-09
  • 第一次建立博客的记录

第一步

  • 下载node.js并安装
  • 下载Git并安装
  • 下载安装Hexo(npm install -g hexo)

第二步 建立GitHub Pages

  • New repository -> 你的Github用户名.github.io 作为仓库名

第三步 创建本地文件

  • 新建一个文件夹，以Blog为例
  • 在文件夹目录下右键选择Git Bash Here，并输入hexo init（初始化博客）
  • 运行 hexo server，此时可以通过输入地址localhost:4000访问博客
